# RTS1 VECTOR-CAM RTS インスタレーション 説明書

## 概要
- 青基調のベクターキャンバス上で、資源採取と歩兵生産を通じて敵城壁の閾値を突破するリアルタイム・インスタレーション。
- 数の論理がすべてであり、臨界数 25 を超えた瞬間に一気に勝利する一方、不足した突撃は銃座迎撃で壊滅する。
- 敵も資源を蓄え迎撃部隊を編成するため、味方と敵の増援バランスを見極めることが鍵となる。

## 遊び方
1. `採取班を編成 (20資源)` で資源生産力を高める。
2. `歩兵を展開 (15資源)` を繰り返し、臨界数 25 に到達させる。
3. `突撃コマンド` で城壁へ前進。臨界到達時のみ勝利できる。
4. 臨界未達の突撃は銃座迎撃により部隊が赤く霧散し、再編成が必要になる。

## システム
- **資源循環**: 初期資源60、採取班は秒間3資源を獲得。敵AIは秒間3.8資源で部隊を増強。
- **城壁と銃座**: 城壁は耐久100。歩兵が臨界未達のまま突撃すると `Condition Red` が発動し、士気低下状態が数秒継続。
- **敵部隊**: 敵は歩兵12体で迎撃射撃を発動。迎撃は味方歩兵を集中的に削り、突撃中なら失敗フェーズへ遷移させる。
- **勝利演出**: 臨界到達後はピラニア群のような侵食アニメで敵基地が崩壊し、金色のフレアが画面に広がる。

## 実行方法
- ブラウザで `index.html` を直接開くか、ローカルサーバーを利用する。
- 例: `python3 -m http.server` を実行し、`http://localhost:8000/RTS1/index.html` にアクセス。

## 開発メモ
- `src/game.js` は全て日本語コメント付きで実装。DOM読み込み後に初期化されるため追加設定は不要。
- ピクセル進捗は `node scripts/pixel-progress.js --percent 100 --output pixel-progress.svg` で生成予定。***
